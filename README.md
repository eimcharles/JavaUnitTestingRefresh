### Java Unit Testing and Test Driven Development refresher and learning ressource.

#### This repository serves as a refresh on core Java unit testing principles using Junit framework, and a learning ressource for AssertJ and Mockito libraries.
--- 

#### Test Driven Development

##### TDD Cycle (repeated for every piece of functionality): 
- **Red**: Write a failing test.
- **Green**: Write just enough code to make the test pass.
- **Refactor**: Clean up code with the safety of passing the test.
